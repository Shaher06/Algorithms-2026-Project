# 🧠 Algorithms Project 2026

## K-th Element of Two Sorted Arrays

---

## 📌 Project Overview

This project is developed as part of the **Algorithms Course (2026)**.
The goal is to solve the **K-th Element of Two Sorted Arrays** problem using two different approaches:

* Non-Recursive Algorithm
* Recursive Algorithm

The project includes full implementation, pseudocode, complexity analysis, and comparison between both approaches.

---

## 🎯 Problem Description

Given two sorted arrays, the task is to find the element that would appear at the **k-th position** in the merged sorted array — without fully merging the arrays.
